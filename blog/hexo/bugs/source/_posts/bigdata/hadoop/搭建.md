## 设置名称

```
172.17.0.2 master
```

编辑`/etc/hostname`

```
master
```

## ssh免密登录

启动ssh服务`/etc/init.d/ssh start`

设置ssh登录配置

编辑 `/etc/ssh/sshd_config` 添加
```xml
PermitRootLogin yes
```

运行生成密码命令：

`ssh-keygen -t rsa`

回车完成密码创建
`cat ~/.ssh/id_rsa.pub >> ~/.ssh/authorized_keys`
`ssh-copy-id root@slave1`

> 使用passwd来重新设置密码
